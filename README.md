EntraVault — Microsoft Entra ID Security Dashboard

EntraVault is a lightweight, browser-based security posture dashboard for Microsoft Entra ID (formerly Azure AD). 
It provides a read-only, real-time assessment of your tenant's security configuration, compliance against global frameworks (CIS, NIST, ISO), and best-practice checks—all without requiring any backend or installation.
🛡️ Security & Privacy First

EntraVault is designed with a "Privacy by Design" approach:

    100% Client-Side: All logic runs in your browser. No data is ever sent to a third-party server.

    Secure Authentication: Uses Microsoft's OAuth2 PKCE (Proof Key for Code Exchange) flow.

    Read-Only: The tool only requests Read permissions. It cannot modify your tenant configuration.

    No Persistence: No tokens or tenant data are stored in permanent storage or databases.

✨ Key Features

    Secure Score Integration: Real-time visualization of your Microsoft Secure Score and improvement actions.

    Compliance Mapping: Automatically maps your configuration against:

        CIS Microsoft 365 Foundations Benchmark

        NIST 800-53

        ISO 27001

        MITRE ATT&CK (Cloud Matrix)

    Conditional Access Audit: Deep-dive into CA policies with an expandable view of inclusions/exclusions.

    Identity Hygiene: Monitoring for Guest user counts, Global Admin sprawl, and MFA method strength.

    Audit Feed: A high-severity event tracker for app consents and role changes.
